# FE 성능 최적화의 이모저모

프론트엔드 성능 최적화 발표 슬라이드 (정적 HTML)

## 실행

```bash
git clone https://github.com/seungjoonH/fe-optimization.git
cd fe-optimization

npx serve . 
```

브라우저에서 `http://localhost:3000` 접속

## 조작


| 입력        | 동작      |
| --------- | ------- |
| `→` / `↓` | 다음 슬라이드 |
| `←` / `↑` | 이전 슬라이드 |
| 좌하단 버튼    | 이전 / 다음 |
| 목차 버튼     | 섹션 이동   |
